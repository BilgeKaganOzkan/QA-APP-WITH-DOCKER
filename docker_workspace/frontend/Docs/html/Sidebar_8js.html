SideBar component for navigating between panels.

This component allows users to toggle between SQL Query and RAG panels. It contains buttons for each panel and a toggle button to show or hide the SideBar. It also handles clicks outside the SideBar to close it.

Parameters
{function}onPanelSelect - Callback function to handle panel selection.

{JSX.Element} The rendered SideBar component.
